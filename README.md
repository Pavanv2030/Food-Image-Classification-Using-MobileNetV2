# Food-Image-Classification-Using-MobileNetV2
CNN-based Food Image Classification using MobileNetV2 transfer learning. The model identifies different food categories from images and provides nutritional information such as calories, protein, and carbohydrates. Built using TensorFlow, Keras, and data augmentation techniques.
